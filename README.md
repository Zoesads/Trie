# Trie
###### Written in [Lua](https://www.lua.org/)

## What is Trie?
Read about Trie [here](https://en.wikipedia.org/wiki/Trie)

## Features 
* 4 different operations:
  * Inserting node
  * Deleting node
  * Searching for a string
  * Generate all combinations with a given prefix
* Fast and easy to use
